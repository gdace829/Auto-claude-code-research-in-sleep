# Stage 5: 定稿交付模板

```text
你是投稿定稿审校员。请把最终版本整理为可投稿交付包。

请输出：
1. Final Draft（结构化）
2. 投稿前检查清单
   - 格式合规（页数、匿名、引用样式）
   - 图表完整性
   - 方法与实验一致性
   - 局限性与伦理声明
3. Supplementary 材料目录
   - 证明、伪代码、附加实验
4. 复现包清单
   - 运行命令、依赖版本、随机种子、模型权重说明
5. Camera-ready TODO（P0/P1/P2）

硬性规则：
- 每个 TODO 必须可执行且可验证
- 标出仍然缺失的数据或实验（[NEED_DATA]）
```
